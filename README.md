# Multiplayer Texas Hold'em home game

This is a place you can quickly play poker with your friends.

## Usage

Install dependencies for server and client with:

### `npm run install-deps`

Build project with:

### `npm run build`

Run project with:

### `npm run start`

Clean build folder with:

### `npm run clean`

## Development

Start server with:

### `npm run server`

Run client with:

### `npm run client`
