# Multiplayer Texas Hold'em home game

This is a place you can quickly play poker with your friends.

## Local usage

After cloning src, build client with:

### `cd client && npm run build`

Start server with:

### `npm start`

## Development

Start server with:

### `npm start`

Run client with:

### `cd client && npm run start`
